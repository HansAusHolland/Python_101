Where is the Space Station? – Leskaarten (NL)

Gebruik deze kaarten om stap voor stap een Python‑programma te maken dat de Internationale Ruimtestation (ISS) op een wereldkaart toont, met live gegevens van openbare web‑API’s.

Stap 1

Wie is er in de ruimte?

Open de webservice die live informatie levert over astronauten in de ruimte:

http://api.open-notify.org/astros.json

In plaats van HTML levert dit adres gegevens (JSON). Je gaat dezelfde service vanuit Python aanroepen.

Python‑code

import requests
import turtle

# URL van de webservice
url = "http://api.open-notify.org/astros.json"

# Aanroepen en JSON laden
astros = requests.get(url, timeout=10).json()

print(astros)  # bekijk de ruwe data

# Nettere weergave
num = astros["number"]
print(f"Mensen in de ruimte: {num}")
people = astros["people"]
for p in people:
    name = p["name"]
    craft = p["craft"]
    print(f"{name} in {craft}")

JSON is een tekstformaat voor gestructureerde data met {sleutel: waarde}‑paren en lijsten [...]. In Python wordt JSON meestal ingelezen als een dict en lijsten.

Stap 2

Waar is het ISS?

Open in je browser de locatiedienst:

http://api.open-notify.org/iss-now.json

Roep een andere service aan vanuit Python en haal de positie van het ISS in breedte‑ en lengtegraad op.

Python‑code

# Locatie van de ISS ophalen
url = "http://api.open-notify.org/iss-now.json"
iss_now = requests.get(url, timeout=10).json()

location = iss_now["iss_position"]
lat = float(location["latitude"])
lon = float(location["longitude"])
print(f"Breedtegraad: {lat}°")
print(f"Lengtegraad: {lon}°")

Breedtegraad (noord/zuid) en lengtegraad (oost/west) vormen samen een coördinaat. Voor Turtle is dit handig als (x, y) = (lon, lat).

Stap 3

Plot het ISS op een kaart

Gebruik Python Turtle om een wereldkaart als achtergrond te laden en zet de wereldcoördinaten gelijk aan lengte‑ en breedtegraden.

Python‑code

# Achtergrond (NASA Visible Earth – kaart 720×360)
screen = turtle.Screen()
screen.setup(1306, 654)
screen.setworldcoordinates(-180, -90, 180, 90)
screen.bgpic("map.gif")

# ISS-icoon
screen.register_shape("iss.gif")
iss = turtle.Turtle()
iss.shape("iss.gif")
iss.setheading(90)
iss.penup()

# Verplaats naar huidige locatie (x=lon, y=lat)
iss.goto(lon, lat)

Let op: normaal noteer je (lat, lon), maar voor (x, y) gebruik je (lon, lat).

Stap 4

Voeg gegevens toe aan je kaart

Schrijf extra informatie op de kaart, zoals het aantal mensen in de ruimte of de ISS‑coördinaten.

Python‑code

# Eén turtle voor alle tekst
writer = turtle.Turtle()
writer.hideturtle()
writer.penup()
writer.color("yellow")

# Schrijf label naast de ISS
writer.goto(lon + 5, lat + 5)
writer.pendown()
writer.write(f"ISS-informatie\nAstronauten: {num}\nLat: {lat:.1f}°\nLon: {lon:.1f}°")
writer.penup()

In code staat "\n" (backslash-n) voor nieuwe regel

Stap 5

Compleet script

Dit script combineert alle stappen: het haalt live data op (astronauten en ISS‑locatie), toont de ISS op de kaart en schrijft een label met coördinaten.

Python-code

import requests
import turtle

# --- Data ophalen ---
astros = requests.get("http://api.open-notify.org/astros.json", timeout=10).json()
iss_now = requests.get("http://api.open-notify.org/iss-now.json", timeout=10).json()

num = astros["number"]
loc = iss_now["iss_position"]
lat = float(loc["latitude"])   # breedtegraad
lon = float(loc["longitude"])  # lengtegraad

# --- Tekstuele feedback in de console ---
print(f"Mensen in de ruimte: {num}")
for p in astros["people"]:
    name = p["name"]
    craft = p["craft"]
    print(f"{name} in {craft}")
print(f"Breedtegraad: {lat}°")
print(f"Lengtegraad: {lon}°")

# --- Tekenvenster en kaart ---
screen = turtle.Screen()
screen.setup(1306, 654)
screen.setworldcoordinates(-180, -90, 180, 90)
screen.bgpic("map.gif")

# --- ISS icoon ---
screen.register_shape("iss.gif")
iss = turtle.Turtle()
iss.shape("iss.gif")
iss.setheading(90)
iss.penup()
iss.goto(lon, lat)

# --- Label (één turtle) ---
writer = turtle.Turtle()
writer.hideturtle()
writer.penup()
writer.color("yellow")
writer.goto(lon + 5, lat + 5)
writer.pendown()
writer.write(f"ISS-informatie\nAstronauten: {num}\nLat: {lat:.1f}°\nLon: {lon:.1f}°")
writer.penup()

# Venster open houden (klik om te sluiten)
screen.exitonclick()
