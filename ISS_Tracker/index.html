Where is the Space Station? – Leskaarten (NL)

Gebruik deze kaarten om stap voor stap een Python‑programma te maken dat de Internationale Ruimtestation (ISS) op een wereldkaart toont, met live gegevens van openbare web‑API’s.

Stap 1

Wie is er in de ruimte?

Open de webservice die live informatie levert over astronauten in de ruimte:

http://api.open-notify.org/astros.json

In plaats van HTML levert dit adres gegevens (JSON). Je gaat dezelfde service vanuit Python aanroepen.

Python‑code

import requests
import turtle

# URL van de webservice
url = "http://api.open-notify.org/astros.json"

# Aanroepen en JSON laden
astros = requests.get(url, timeout=10).json()

print(astros)  # bekijk de ruwe data

# Nettere weergave
num = astros["number"]
print(f"Mensen in de ruimte: {num}")
people = astros["people"]
for p in people:
    name = p["name"]
    craft = p["craft"]
    print(f"{name} in {craft}")

Wat moet ik hier doen?

  1. Roep de API aan met requests.get(...).json().
  2. Lees number en people uit de dict.
  3. Print elk item als naam in ruimtevaartuig.

Succescheck

  • Console toont Mensen in de ruimte: <getal>.
  • Je ziet minstens één regel met een naam en craft.

Veelgemaakte fout → fix

Fout: KeyError: 'people'
Fix: Je gebruikt waarschijnlijk de rauwe tekst. Vergeet .json() niet of print eerst type(astros).
Fout: Timeout / netwerkfout
Fix: Voeg timeout=10 toe en probeer later opnieuw.

Verwachte output (voorbeeld)

Mensen in de ruimte: 7
Jasmin Moghbeli in ISS
...

Conceptueel: Waarom JSON?

JSON is taal‑agnostisch en compact. In Python wordt het automatisch een dict (sleutels) met lijsten. Dit maakt doorlopen met for eenvoudig.

Differentiatie

  • Uitdaging: Sorteer op craft en tel per voertuig.
  • Stap‑voor‑stap nodig? Print eerst astros.keys(), daarna len(astros['people']).

⏱️ Timer & hulp

6–8 min. Zit je na 4 min vast? Controleer: (1) internet, (2) .json(), (3) sleutelnaam exact gespeld.

Stap 2

Waar is het ISS?

Open in je browser de locatiedienst:

http://api.open-notify.org/iss-now.json

Roep een andere service aan vanuit Python en haal de positie van het ISS in breedte‑ en lengtegraad op.

Python‑code

# Locatie van de ISS ophalen
url = "http://api.open-notify.org/iss-now.json"
iss_now = requests.get(url, timeout=10).json()

location = iss_now["iss_position"]
lat = float(location["latitude"])
lon = float(location["longitude"])
print(f"Breedtegraad: {lat}°")
print(f"Lengtegraad: {lon}°")

Wat moet ik hier doen?

  1. Roep iss-now.json aan.
  2. Lees iss_position en converteer strings naar float voor lat en lon.
  3. Print beide met graden‑symbool.

Succescheck

  • Console toont twee getallen binnen bereik: lat ∈ [−90, 90], lon ∈ [−180, 180].
  • Geen ValueError bij conversie.

Veelgemaakte fout → fix

Fout: TypeError: float(None)
Fix: Print eerst iss_now en check pad: ['iss_position']['latitude'].

Waarom float?

De API levert getallen als tekst. Voor rekenwerk en tekenen met Turtle heb je numerieke types nodig.

Differentiatie

  • Uitdaging: Format output op 4 decimalen en voeg noorder/zuider, ooster/wester toe.
  • Stap‑voor‑stap: Doe eerst alleen print(iss_now); haal daarna stap voor stap velden eruit.

⏱️ Timer & hulp

5–7 min. Vast? Controleer paden en dat je float(...) gebruikt.

Stap 3

Plot het ISS op een kaart

Gebruik Python Turtle om een wereldkaart als achtergrond te laden en zet de wereldcoördinaten gelijk aan lengte‑ en breedtegraden.

Python‑code

# Achtergrond (NASA Visible Earth – kaart 720×360)
screen = turtle.Screen()
screen.setup(1306, 654)
screen.setworldcoordinates(-180, -90, 180, 90)
screen.bgpic("map.gif")

# ISS-icoon
screen.register_shape("iss.gif")
iss = turtle.Turtle()
iss.shape("iss.gif")
iss.setheading(90)
iss.penup()

# Verplaats naar huidige locatie (x=lon, y=lat)
iss.goto(lon, lat)

Wat moet ik hier doen?

  1. Maak een Turtle‑Screen en zet wereldcoördinaten op (−180, −90, 180, 90).
  2. Laad map.gif als achtergrond en registreer iss.gif.
  3. Plaats de turtle op (lon, lat).

Succescheck

  • Je ziet een wereldkaart met het ISS‑icoon ongeveer op de juiste plek.
  • Vensterafmeting ≈ 1306×654 en vervormt de kaart niet.

Veelgemaakte fout → fix

Fout: Plaatsing omgedraaid (lat, lon).
Fix: Gebruik goto(lon, lat).
Fout: Bestanden niet gevonden.
Fix: Zorg dat map.gif en iss.gif in dezelfde map staan als je script.

Conceptueel: Projectie

Door setworldcoordinates(-180, -90, 180, 90) te gebruiken, map je pixel‑x op lengtegraad en pixel‑y op breedtegraad.

Differentiatie

  • Uitdaging: Laat het icoon elke 5 s naar nieuwe coördinaten bewegen.
  • Stap‑voor‑stap: Test eerst zonder iconen; gebruik een standaardvorm turtle.Turtle() met shape('circle').

⏱️ Timer & hulp

7–10 min. Vast? Test goto(0,0) (snijpunt evenaar/meridiaan) om je schaal te checken.

Let op: normaal noteer je (lat, lon), maar voor (x, y) gebruik je (lon, lat).

Stap 4

Voeg gegevens toe aan je kaart

Schrijf extra informatie op de kaart, zoals het aantal mensen in de ruimte of de ISS‑coördinaten.

Python‑code

# Eén turtle voor alle tekst
writer = turtle.Turtle()
writer.hideturtle()
writer.penup()
writer.color("yellow")

# Schrijf label naast de ISS
writer.goto(lon + 5, lat + 5)
writer.pendown()
writer.write(f"ISS-informatie\nAstronauten: {num}\nLat: {lat:.1f}°\nLon: {lon:.1f}°")
writer.penup()

Wat moet ik hier doen?

  1. Maak één schrijvende turtle voor labels.
  2. Kies een contrasterende color().
  3. Schrijf een meerregelig label naast het ISS met \n.

Succescheck

  • Label verschijnt dicht bij de ISS‑positie en overlapt het icoon niet.
  • Tekst gebruikt afronding (bijv. {lat:.1f}).

Veelgemaakte fout → fix

Fout: Label blijft tekenen over de kaart.
Fix: Gebruik penup() vóór verplaatsen en na write().

Conceptueel: Nieuwe regel

"\n" creëert een regelbreuk binnen dezelfde write()‑aanroep.

Differentiatie

  • Uitdaging: Voeg lokale tijd toe met datetime.
  • Stap‑voor‑stap: Begin met één regel tekst; breid daarna uit.

⏱️ Timer & hulp

4–6 min. Vast? Schakel tijdelijk een felle kleur in (bv. wit) om zichtbaar te testen.

Stap 5

Compleet script

Dit script combineert alle stappen: het haalt live data op (astronauten en ISS‑locatie), toont de ISS op de kaart en schrijft een label met coördinaten.

Python-code

import requests
import turtle

# --- Data ophalen ---
astros = requests.get("http://api.open-notify.org/astros.json", timeout=10).json()
iss_now = requests.get("http://api.open-notify.org/iss-now.json", timeout=10).json()

num = astros["number"]
loc = iss_now["iss_position"]
lat = float(loc["latitude"])   # breedtegraad
lon = float(loc["longitude"])  # lengtegraad

# --- Tekstuele feedback in de console ---
print(f"Mensen in de ruimte: {num}")
for p in astros["people"]:
    name = p["name"]
    craft = p["craft"]
    print(f"{name} in {craft}")
print(f"Breedtegraad: {lat}°")
print(f"Lengtegraad: {lon}°")

# --- Tekenvenster en kaart ---
screen = turtle.Screen()
screen.setup(1306, 654)
screen.setworldcoordinates(-180, -90, 180, 90)
screen.bgpic("map.gif")

# --- ISS icoon ---
screen.register_shape("iss.gif")
iss = turtle.Turtle()
iss.shape("iss.gif")
iss.setheading(90)
iss.penup()
iss.goto(lon, lat)

# --- Label (één turtle) ---
writer = turtle.Turtle()
writer.hideturtle()
writer.penup()
writer.color("yellow")
writer.goto(lon + 5, lat + 5)
writer.pendown()
writer.write(f"ISS-informatie\nAstronauten: {num}\nLat: {lat:.1f}°\nLon: {lon:.1f}°")
writer.penup()

# Venster open houden (klik om te sluiten)
screen.exitonclick()

Wat moet ik hier doen?

  1. Combineer alle stappen in één script.
  2. Houd het venster open met screen.exitonclick().

Succescheck

  • Script runt zonder fouten en toont kaart, icoon en label.
  • Console toont astronautenlijst en coördinaten.

Veelgemaakte fout → fix

Fout: Scherm sluit direct.
Fix: Voeg screen.exitonclick() toe.
Fout: Onjuiste imports.
Fix: Zorg dat import requests en import turtle bovenaan staan.

Robuustheid (optioneel)

try:\n    astros = requests.get(\"http://api.open-notify.org/astros.json\", timeout=10).json()\nexcept Exception as e:\n    print(\"Kon astronautendata niet laden:\", e)\n    astros = {\"number\": 0, \"people\": []}\n

⏱️ Timer & hulp

8–12 min. Klaar? Voeg een refresh toe om de positie te updaten.
